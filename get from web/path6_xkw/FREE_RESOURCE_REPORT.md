## 4. 试卷特色

### 同步专区
- 按教材章节组织
- 包含：单元测试、课时练习、章节复习
- 适合日常教学同步使用

### 备考专区
- 按阶段考试类型组织
- 筛选维度：papertype（考试类型）、termid（学期）、paperyear（年份）、province_id（省份）
- 适合期中期末复习

### 高考专区
- 高考真题 + 各地模拟题
- 筛选维度：papertype（真题/模拟/复习/压轴）、paperyear（年份）、province_id（省份）
- 覆盖全国34个省市的模拟卷

## 5. 题目提取可行性

### SSR 方式（requests，无需浏览器）
- 题目文本：100% 可提取
- 选项：100% 可提取
- 知识点标注：~90% 可提取（部分以图片形式存储）
- 难度：100% 可提取
- 题目来源：100% 可提取
- 答案：0%（需登录）
- 解析：0%（需登录）

### Playwright 方式（需浏览器 + 登录）
- 当 Path 1 的 storage-state.json 可用且 chrome_freed 信号出现时
- 可模拟点击"显示答案"获取答案图片
- 每日限额 30 次答案查看
- 答案以图片 URL 形式存储在 webshot.zujuan.com

## 6. 与 Path 1 的互补关系

| 维度 | Path 1（知识点搜题） | Path 6（整卷拆分） |
|------|---------------------|---------------------|
| 获取方式 | 按知识点筛选 → 逐题抓取 | 按试卷浏览 → 整卷拆分 |
| 题目组织 | 按知识点聚合 | 按试卷结构组织（有题号、题型分区） |
| 题型覆盖 | 可指定单一题型 | 一张试卷包含全部题型 |
| 知识点标注 | 已知知识点（搜索条件） | 每题自带知识点标注 |
| 难度信息 | 已知难度（搜索条件） | 每题自带难度等级 |
| 适用场景 | 针对性练习 | 完整试卷模拟 |
| 数据格式 | 截图为主 + OCR文字 | HTML文字提取（无截图） |
| 答案 | 可获取（登录后） | 当前不可获取（未登录） |

## 7. 建议

1. **短期**：利用 SSR 方式批量获取试卷题目文本，构建题库索引
2. **中期**：等 Path 1 完成登录后，复用其 storage-state.json 补充答案数据
3. **长期**：两个路径的数据合并，形成完整的"题目+知识点+难度+答案"题库
